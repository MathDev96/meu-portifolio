Admin Dashboard

Criei, junto à equipe voluntária do projeto, um admin dashboard utilizando React com MaterialUI para o front-end e Java com Spring Boot para o back-end.

Com esse projeto, proporcionamos à ONG um gerenciamento eficiente dos dados das doações, permitindo a geração de relatórios com estatísticas sobre os valores arrecadados e o número de doadores.

Além disso, o sistema possibilita o envio de agradecimentos por e-mail ou notificações personalizadas. O dashboard inclui gráficos de arrecadação mensal, total de doações e comparações com meses anteriores, além de uma análise do perfil dos doadores, identificando padrões de doação.

Mulheres que Reagem

Me reuni com uma equipe de desenvolvedores e designers voluntários para criar um site completo para a ONG, dedicado ao projeto Mulheres que Reagem, que apoia e empodera mulheres.

Construimos um site dinâmico, com várias seções que vão desde informações sobre a ONG e seus projetos até áreas para doações. No front-end, estamos usando React, JavaScript, HTML, CSS e Figma para garantir uma experiência visual impactante e totalmente responsiva.

No back-end, desenvolvemos uma API robusta com Java e Spring Boot para gerenciar as doações e fornecer uma infraestrutura segura e eficiente para a ONG.
